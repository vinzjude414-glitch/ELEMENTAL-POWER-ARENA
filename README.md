Create a Roblox game called "Elemental Powers Arena".

Game Description:
Players join and choose between Fire and Lightning elements. After choosing, they are teleported into a battle arena where they fight other players.

Features:
- Element selection GUI (Fire 🔥, Lightning ⚡)
- Arena map (floating island style)
- Combat system with abilities

Fire Skills:
- Fireball: shoots a fast projectile that deals 20 damage
- Flame Aura: damages nearby enemies over time

Lightning Skills:
- Dash Strike: player dashes forward quickly and damages enemies
- Thunder Stun: stuns enemies for 2 seconds

Systems:
- Leaderboard (kills and coins)
- XP and leveling system
- Respawn system after death

Extra:
- Add simple UI buttons for abilities
- Add cooldown system for each skill
- Add basic sound effects and particles

Make everything simple and beginner-friendly.
